# wync-cookbook
===============

This installs Linux version of Lync.


Attributes
----------
* `wync::default'` - installs the package

## Usage

### wync::default

Include `wync` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[wync::default]"
  ]
}
```
